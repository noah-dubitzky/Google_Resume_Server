Pixel Battle

a fast-paced 2D platformer hack-and-slash game using JavaScript and the Phaser framework. The game features vibrant pixel-art environments where players navigate challenging levels filled with enemies and obstacles. The project uses complex classes and functions to develope characters and define game mechanics.
